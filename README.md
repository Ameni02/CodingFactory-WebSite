## ğŸ“Š AI Workflow

  

PFESpace implements a sophisticated AI workflow:

  

### 1. CV Analysis

The system analyzes CVs using a weighted scoring algorithm:

-  **Education (15%)**: Academic background and qualifications

-  **Experience (20%)**: Relevant work experience and internships

-  **Skills (30%)**: Match between skills and project requirements

-  **Project Match (15%)**: Relevance to the specific project

-  **Field Match (10%)**: Alignment with the project's domain

-  **Title Match (10%)**: Keywords matching the project title

  

Applications scoring 60% or higher are automatically accepted, 50-59% are placed in pending for manual review, and below 50% are automatically rejected.

  

### 2. Project Matching

The SBERT-based matching algorithm:

- Extracts key skills from the CV

- Compares them with project requirements

- Evaluates field and domain relevance

- Ranks projects by compatibility score

  

### 3. Cover Letter Generation

AI-powered cover letter creation:

- Contextualizes CV information with project details

- Generates personalized content highlighting relevant skills

- Formats according to professional standards

- Uses Hugging Face models for natural language generation

  

### 4. Content Analysis

Comprehensive content verification:

-  **Plagiarism Detection**: Identifies similarities with existing documents

-  **AI Content Detection**: Recognizes AI-generated text

-  **Detailed Feedback**: Provides specific improvement suggestions

## ğŸ”„ API Endpoints

  

### Project Management

-  `GET /api/pfe/projects` - Get all projects

-  `GET /api/pfe/projects/{id}` - Get project by ID

-  `POST /api/pfe/projects` - Create new project

-  `PUT /api/pfe/projects/{id}` - Update project

-  `DELETE /api/pfe/projects/{id}` - Delete project

  

### Application Management

-  `GET /api/pfe/applications` - Get all applications

-  `POST /api/pfe/applications` - Submit application

-  `GET /api/pfe/applications/{id}` - Get application details

-  `PUT /api/pfe/applications/{id}/status` - Update application status

  

### AI Services

-  `POST /api/pfe/match-projects` - Match CV to projects

-  `POST /api/pfe/generate-cover-letter` - Generate cover letter

-  `POST /api/pfe/projects/{id}/analyze-cv` - Analyze CV for specific project

-  `POST /api/pfe/deliverables/{id}/analyze-content` - Analyze deliverable content

  

## â“ FAQ

  

**Q: How does the CV analysis system work?**

A: The system analyzes CVs using a weighted scoring algorithm that evaluates education (15%), experience (20%), skills (30%), project match (15%), field match (10%), and title match (10%). Applications scoring 60% or higher are automatically accepted.

  

**Q: Can students apply to multiple projects?**

A: Yes, students can apply to multiple projects, but each application requires a separate submission with a tailored cover letter.

  

**Q: How is plagiarism detected in deliverables?**

A: The system uses a combination of n-gram analysis and semantic comparison to detect similarities between submitted documents and a reference database.

  

**Q: Is the platform integrated with other university systems?**

A: Yes, PFESpace integrates with the Gestion_User microservice to access student and supervisor information.

  

**Q: How accurate is the AI content detection?**

A: The AI content detection combines local pattern matching (30% weight) and Hugging Face API analysis (70% weight) to achieve high accuracy in identifying AI-generated content.
